Hackathon Dashboard

Ruby on Rails, Postgres, a whole load of gems

Originally written as a dashboard for HackUMass, this SaaS is built so groups wishing to organize their own hackathon can have a dashboard that organizes mentors, attendees, events, and much more. We're currently preparing to launch it open source and documenting the software so anyone can use it. Check it out soon, its sponsored by Github!

WTF UMass

Node, ExpressJS, Vue, MongoDB

What to fix (WTF) UMass is a website written in Node, Vue, and MongoDB where student can submit feedback anonymous to their peers about UMass for their Student Government Assocation to work on. Other students can come and vote for the issues that they want to see fixed on campus, and get status updates on their presented ideas. Check it out at wtfumass.com.

ARG!

HTML, CSS, JQuery

This simple reaction time game was written for researchers to be able to create a scripted environment where a subject can compete against an unknown opponent in a scripted environment. Researchers have complete control over who wins and loses and get a CSV of the actions of the player at the end of the trial. See the code here and play five rounds against the computer here. Watch out! They're pretty good.
